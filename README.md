# RC-car

## 仕様

* 使用ハード：Arduino UNO R4 WiFi<br>
* 開発環境：Arduino IDE<br>
* 使用言語：C++<br>
* 操作方法：UDPを用いた無線通信による制御<br>
* 搭載モジュール：赤外線センサ*1、(カメラ)*2<br>


*1：前方に障害物を検知した場合停止するために搭載。<br>
*2：スマートフォンを使用したライブ配信用カメラ。Arduinoとは接続していない。<br>
